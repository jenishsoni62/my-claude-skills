# My Claude Skills

A collection of custom skills for [Claude AI](https://claude.ai).

---

## Excel Report Generator

Automatically transforms any CSV file into a professional, fully-formatted Excel report.

### What it generates
- **Dashboard** — KPI cards with total revenue, profit, margin, units sold
- **Data sheet** — styled table with color-coded headers and alternating rows
- **Charts** — bar chart, line trend chart, and pie chart by region
- **Analysis** — financial formulas: totals, averages, max, min, growth %

### How to install
1. Download `excel-report-generator.skill`
2. Go to **Claude.ai → Settings → Customize → Skills**
3. Click the **+** button and upload the file
4. Done!

### How to use
1. Upload any `.csv` file in Claude chat
2. Type: `generate an Excel report from this CSV`
3. Download the generated `.xlsx` file

### Example prompt
> "Generate an Excel report from this CSV with a dashboard and charts"
